采用零数据保留策略，你的创作提示以及生成的图像，永远不会存储在我们的服务器上，充分保障隐私安全 。

三、使用指南
访问工具：打开浏览器，进入工具网页（待补充实际访问地址 ）。
输入创意：在文本输入框中，详细描述你想要生成的图像内容，比如 “星空下的日式庭院，樱花飘落” ，也可结合风格要求补充描述，像 “以油画风格呈现” 。
